# Отчёт по практике: Лазерная резка органайзеров из фанеры

## Цель работы
Освоить технологию лазерной резки на практике, спроектировать и изготовить три функциональных органайзера.

## Изготовленные изделия

### Органайзер 1: Стакан для канцелярии на фанерном основании
 **Назначение:** Для хранения ручек, карандашей и других письменных принадлежностей
 **Материал:** Фанера 6 мм + 3D-печатный стакан
 **Описание процесса:**
    1.  Стакан для канцелярии был изготовлен на 3D-принтере (предоставлен преподавателем)
    2.  Фанерное основание вырезано на лазерном станке
    3.  Параметры резки фанеры: **мощность 98%**, **скорость 1.1%**
    4.  Стакан прикреплен к фанерному основанию с помощью четырех болтов М3х30, и ножками для стенки сделанные такэе из фанеры 
 **Итог:** Устойчивая конструкция, стакан надежно закреплен на основании

### Органайзер 2: Органайзер для клеев разных размеров
 **Назначение:** Упорядочивание и хранение клеев различных размеров
 **Материал:** Фанера 6 мм
 **Описание процесса:**
    1.  Разработка конструкции с отделениями под разные размеры клеев
    2.  Лазерная резка всех деталей из фанеры 6 мм
    3.  Параметры резки: **мощность 98%**, **скорость 1.1%**
    4.  Сборка деталей в готовую конструкцию
 **Итог:** Органайзер удобно размещает клеи разных размеров, конструкция устойчива, так как была закреплена с помощью шести болтов М3х30
